# Projeto para testar o Chakra UI

## O que é o Chakra UI?

O Chakra UI é um framework de componentes que permite criar interfaces de usuário com React.

Para saber mais sobre o Chakra UI, acesse o site oficial: [https://chakra-ui.com/](https://chakra-ui.com/)

## Como rodar o projeto

Para rodar o projeto, basta clonar o repositório e executar o comando `yarn` para instalar as dependências.

Após isso, basta executar o comando `yarn dev` para rodar o projeto.

O servidor estará disponível em [http://localhost:51](http://localhost:5173)
